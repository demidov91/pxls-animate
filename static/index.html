There is one more feature not represented on the page: you can specify start and end time for gif files. Time is in FET timezone (UTC+03:00)

GIF API: pxls.pautuzin.by/gif?start-point=0-677&end-point=363-1405&start=2017-04-11-1300&end=2017-04-16-1400

Image API (I don't have data earlier than this snapshot): pxls.pautuzin.by/im?start-point=0-0&end-point=1999-1999&time=2017-04-11-1300
